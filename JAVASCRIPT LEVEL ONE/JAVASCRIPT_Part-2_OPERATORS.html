The comparsion and logical operator with Javascript


Javascript uses "===" while we compare different data types not values in console. Similarly for comparing different data types not values objects "!==" is used.


This operator allow us to begin to add logic to our Javascript code


Some weird behaviour in console while using operators are: if we write null == undefined (true) null === undefined (false) NaN == NaN (false)

Logical Operators

  • AND operator

    • && in Javascript.

    Ex.- (1===2) && (2===2) returns false

  • OR operator

    • || in Javascript.

    Ex.- (1===2) && (2===2) returns true

  • NOT operator

    • " ! "in Javascript.

    Ex.- !((1===2) && (2===2)) returns true
