ARRAYS

Key features of Javascript Arrays:

  • JavaScript arrays are resizable and can contain a mix of different data types
  • JavaScript arrays are not associative arrays and so, array elements cannot be accessed using arbitrary strings as indexes, but must be accessed using nonnegative integers (or their respective string form) as indexes.
  • JavaScript arrays are zero-indexed
  • JavaScript array-copy operations create shallow copies rather than deep copies

Some of the Array methods.

  • concat()

    • array1.concat(array2);

    return a new array containing array1 and array2

  • array.flat(Infinity)

    • returns entire flatten array

  • array.indexOf(searchElement)

    • return index of the searchElement

  • array.lastIndexOf(searchElement)

    • return the last index of the searchElement

  • array.push(item)

    • pushes element to the end of the array

  • array.pop(item)

    • pops last element of the array

  • array.sort()

    • sorts the array in ascending order.
